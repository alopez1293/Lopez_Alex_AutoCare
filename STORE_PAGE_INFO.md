# Store Page Information

## Project Title

AutoCare Log

## Short Description

AutoCare Log helps drivers track their vehicles, mileage, service history, and upcoming maintenance in one simple mobile app.

## Full Description

AutoCare Log is a mobile vehicle maintenance tracker designed to help users keep better records for their personal vehicles. The app allows users to view a garage dashboard, add vehicles, track mileage, monitor upcoming oil change reminders, and log service events such as oil changes, brake work, tire rotations, and other maintenance.

The app also includes driving condition information from a weather API to help users think about how current weather may affect vehicle care. Users can select a photo from their device photo library and attach it to a vehicle card, making the garage screen feel more personal and realistic.

AutoCare Log is designed as a simple, practical tool for organizing vehicle information and building a clear maintenance history over time.

## Target Audience

AutoCare Log is designed for everyday drivers, students, families, and vehicle owners who want a simple way to track basic maintenance records without using a complicated spreadsheet or paper notebook.

## Main Features

- View a garage dashboard with saved vehicles
- Add a new vehicle with mileage, VIN, oil change mileage, and notes
- View vehicle details and maintenance status
- Track whether a vehicle is close to its next oil change
- Log service events with date, service type, odometer reading, cost, and notes
- View service history for each vehicle
- Select and display a vehicle photo using the device photo library
- View current driving condition information from a weather API
- See helpful empty states, validation messages, and loading messages

## Device Permissions Required

- Photo Library / Media Library permission

## Why Permissions Are Needed

Photo Library permission is needed so the user can choose a vehicle photo from their device and display it on the vehicle card in the garage. The app does not require the user to select a photo, but the permission is required when the user chooses to use the vehicle photo feature.

## Category

Productivity / Lifestyle

## Age Rating Considerations

AutoCare Log does not include social networking, user-generated public content, mature content, gambling, violence, or location tracking. The app is safe for a general audience, but it is mainly designed for people who own, drive, or help maintain vehicles. Because of that, the practical target user is more likely to be a teen or adult rather than a young child, even though the app content itself is not inappropriate for younger users.
